# Changelog

All notable changes to the Cloudbeds Zapier Integration will be documented in this file.

## [1.0.0] - 2024-01-XX

### Added
- Initial release of Cloudbeds Zapier Integration
- OAuth2 authentication support
- 5 Triggers:
  - New Reservation
  - New In-House Guest
  - New Checked-Out Guest
  - New Cancellation
  - New Guest
- 4 Actions:
  - Create Reservation
  - Update Reservation
  - Cancel Reservation
  - Create Guest
- 2 Searches:
  - Find Reservation
  - Find Guest
- Dynamic fields for properties and room types
- Comprehensive error handling and validation
- Sample data for all triggers
- Helper utilities for date formatting and data validation
- Complete documentation (README, SETUP, EXAMPLES)

### Features
- Full Cloudbeds API v1.1 support
- Automatic token refresh
- Input validation
- Error handling with descriptive messages
- Dynamic dropdown fields for better UX
- Comprehensive example workflows
