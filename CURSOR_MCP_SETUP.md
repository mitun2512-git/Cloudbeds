# Using Cloudbeds with Zapier MCP in Cursor

Since you've connected Zapier MCP to Cursor, here's how to access Cloudbeds through it.

## Step 1: Connect Cloudbeds to Your Zapier Account

Before you can use Cloudbeds through MCP, you need to connect it in Zapier:

1. **Go to Zapier:**
   - Visit [zapier.com/apps/cloudbeds](https://zapier.com/apps/cloudbeds)
   - Or search for "Cloudbeds" in Zapier apps

2. **Connect Your Account:**
   - Click "Connect Cloudbeds + Zapier"
   - Sign in with your Cloudbeds credentials
   - Authorize Zapier to access your Cloudbeds data
   - Your account is now connected!

## Step 2: Add Cloudbeds Tools to Your MCP Server

1. **Go to Your MCP Server:**
   - Visit [mcp.zapier.com](https://mcp.zapier.com/)
   - Select your MCP server (the one connected to Cursor)

2. **Add Cloudbeds Actions:**
   - Go to "Tools" or "Actions" section
   - Search for "Cloudbeds"
   - Add these recommended tools:
     - ✅ **Create Reservation** - Create new reservations
     - ✅ **Find Reservation** - Search for reservations
     - ✅ **Get Reservations** - List reservations
     - ✅ **Update Reservation** - Modify reservations
     - ✅ **Cancel Reservation** - Cancel reservations
     - ✅ **Create Guest** - Add guest profiles
     - ✅ **Find Guest** - Search for guests
     - ✅ **Get Guests** - List guests

3. **Configure Each Tool:**
   - Select your connected Cloudbeds account
   - Set any default values if needed
   - Save the configuration

## Step 3: Verify MCP Connection in Cursor

1. **Check MCP Status:**
   - In Cursor, you should see MCP tools available
   - Look for Zapier-related tools in your command palette

2. **Test the Connection:**
   - Try asking me (the AI) to use Zapier tools
   - Example: "Can you check what Zapier tools are available?"

## Troubleshooting

### "Cloudbeds not found" or "No tools available"
- **Solution**: Make sure you've connected Cloudbeds in Zapier first
- Go to [zapier.com/apps/cloudbeds](https://zapier.com/apps/cloudbeds) and connect your account
- Then add Cloudbeds tools to your MCP server

### "Authentication failed"
- **Solution**: Reconnect your Cloudbeds account in Zapier
- The OAuth token may have expired
- Go to Zapier → Connected Accounts → Reconnect Cloudbeds

### "Tool not available"
- **Solution**: Check that you've added the tool to your MCP server
- Go to [mcp.zapier.com](https://mcp.zapier.com/) → Your Server → Tools
- Make sure Cloudbeds tools are enabled

### MCP connection not working in Cursor
- **Solution**: Verify MCP configuration in Cursor settings
- Check that your MCP server endpoint is correct
- Restart Cursor if needed
